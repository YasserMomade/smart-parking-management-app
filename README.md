# Smart Parking Management App

A smart parking management and reservation system designed to simplify parking operations for both administrators and users.  
The application supports permanent parking areas as well as event-based parking, allowing users to book parking spots, view availability, and make payments directly from the mobile app.

This project was developed as a team-based academic project, with a strong focus on organization, real-world use cases, and scalability.
